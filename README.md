# notes-web-cns

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立编写的 HTML 页面，涵盖笔记、工具页面、文档展示等内容。每个页面均为独立的静态文件，不依赖任何后端服务或特定域名环境，可直接在浏览器中打开使用。

## 目录说明

- `pages/`：存放所有独立 HTML 页面文件，每个文件一个页面。
- `assets/`：存放页面所需的公共资源，如图片、样式表、脚本等。
- `index.html`：仓库首页，列出所有已归档页面的简单导航（可选）。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，内部可能引用 `assets/` 下的资源。
- 页面命名采用有意义的英文或拼音，便于识别内容主题。
- 归档页面可能不定期更新或新增，不保证所有页面长期可用或兼容最新浏览器版本。

## 维护说明

- 本仓库为个人归档用途，不接受外部直接推送修改。
- 如有建议或问题，欢迎通过 Issues 提出。
- 页面内容仅供参考学习，不构成任何形式的建议或承诺。
